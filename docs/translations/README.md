# Translations

This directory contains translations of the project documentation for non-English speaking users.

**Note:** The English versions in the project root are the authoritative sources. These translations are provided as a convenience and may not always be up to date.

## Available Translations

- **简体中文 (Simplified Chinese)**
  - [README_ZH.md](README_ZH.md) - 项目概述
  - [DEVELOPMENT_ZH.md](DEVELOPMENT_ZH.md) - 开发指南

## Keeping Translations in Sync

If you update any of these translations, please also update the corresponding English files in the project root to ensure consistency.

For adding new translations, please refer to the project's [CONTRIBUTING.md](../../CONTRIBUTING.md) guidelines.
